    Lightkurve Data

    This dashboard uses the Lightkurve project to access, clean, and visualize NASA Kepler and TESS light curves. Lightkurve provides a Python API for downloading target pixel files (TPFs) and light curves, performing detrending (e.g., flatten()), stitching sectors, and phase-folding data around transit times. In this app, generated GIFs summarize a 3D orbit schematic and the phased light curve using Lightkurve-processed flux measurements.

    Library: lightkurve.github.io/lightkurve
